# locof - Lines of Code Of
`locof` is a streamlined, Rust-based CLI tool designed to efficiently count the lines of code within a project directory. Inspired by [cloc](https://github.com/AlDanial/cloc).

This project is currently under development and serves as both a practical tool for developers and a learning Rust's capabilities for CLI development.

## Usage
```bash
$ locof <project_directory>
```
